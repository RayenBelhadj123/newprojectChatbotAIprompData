# Updated LaTeX Report Project - US Housing Intelligence Platform

Main file to compile on Overleaf:
- rapport.tex

This updated package preserves the original report structure and adds the new project features:
- OLAP analysis
- AI chatbot
- Prompt chatbot
- Reinforcement learning
- Unsupervised learning
- Streamlit deployment
- Paper review and real-market comparison
- Experiment tracking
- Model registry
- Production-readiness checks
- Business impact analysis

Included source/data files:
- streamlit_app.py
- streamlit_app(oldversion).py
- us_home_price_analysis_2004_2024.csv

How to use:
1. Upload all files to Overleaf.
2. Set rapport.tex as the main file.
3. Compile with pdfLaTeX.
4. Replace placeholder screenshots with your real dashboard screenshots if required.
5. To run the app locally: streamlit run streamlit_app.py
