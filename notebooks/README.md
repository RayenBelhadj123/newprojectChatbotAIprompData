# Notebooks

Use these folders to keep analysis notebooks organized by workflow stage.

| Folder | Purpose |
| --- | --- |
| `1-data` | Data extraction and cleaning. |
| `2-exploration` | Exploratory data analysis. |
| `3-analysis` | Statistical analysis and hypothesis checks. |
| `4-feat_eng` | Feature engineering. |
| `5-models` | Model training and evaluation. |
| `6-interpretation` | Model interpretation. |
| `7-deploy` | Deployment experiments. |
| `8-reports` | Reporting and storytelling. |

Suggested naming pattern:

```text
01-initial_exploration-2026_04_25.ipynb
```
